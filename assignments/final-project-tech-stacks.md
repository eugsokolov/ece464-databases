# Suggested Tech Stacks

- **Django**
  - Full-featured ORM + Web + UI templates
  - More strict framework
- **Psycopg2** (DB driver)
- **SQLAlchemy** + Web framework
- **Alembic** (migrations)
- **Argparse** (simple CLI)
